# Yazi cheatsheet

- `~`: Show help
- `q`: Quit
- `a`: Create file
- `.`: Toggle hidden files
- `,`: Change sorting (opens menu)
- `g`: Go to directory (opens menu)

## Tabs

- `t`:     New tab
- `<C-q>`: Quit tab
- `[`/`]`: Prev/next tab
- `{`/`}`: Move current tab to the left/right
- `0`, `1`, `2`, etc: Switch to nth tab (0-indexed)
